# Come Mingle, Bring Your Singles

Landing page for a friends-of-friends mingle night at Schoolyard Beer Hall, Denver — Friday June 19, 5–9 PM. 21+, no cover, 100 spots, Fresh Barbershop on-site.

## What's in here

A single static `index.html` — all CSS and JavaScript inline, no build step, no dependencies. Open it in a browser to preview, or deploy as-is.

## Deploy

Hosted on Vercel as a static site. Pushes to `main` auto-deploy via Vercel's GitHub integration.

## Edit

Everything lives in `index.html`. Common edits:

- **RSVP link** — search for `Replace this link with your Partiful or Eventbrite URL` and swap the placeholder for your real RSVP URL.
- **Date/time** — search for `June 19` (and the countdown target near the bottom: `2026-06-19T17:00:00-06:00`).
- **Venue** — search for `Schoolyard Beer Hall`.
- **Social handles** — search for `bringyoursingles` and `fresh_barbershop_denver` in the footer.
